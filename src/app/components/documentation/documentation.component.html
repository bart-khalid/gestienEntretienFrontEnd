L' admin a tous droits à gérer plusieurs actions; Un admin peut ajouter , modifier ou supprimer les différents choses suivantes: Matériel, Locale, Véhicule, ainsi qu'affecter un matériel à une locale, en plus de ça il peut sauvegarder et gérer les différents type des bons. Enfin un admin peut sauvegarder les prestations qui se déroulent au sein de la Facultés et de les gérer aussi, comme il a aussi le droit de gérer les reclamations qui sont déclarer par les employés. Un admin est le seul responsable de la gestion des utilisateurs càd il peut ajouter ou supprimer ou modifier un utilisateur. Un employé peut seulement faire des reclamations, ainsi le suivre de l'état de ces reclamations. Il a aussi le droit de modifier ou supprimer les reclamations qu'il a déjà faits.
